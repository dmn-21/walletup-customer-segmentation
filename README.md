# Bank Customer Segmentation Analysis

Customer segmentation project developed in RStudio and Quarto using clustering techniques and exploratory data analysis.

## Live Project
🔗 https://dmn-21.github.io/walletup-customer-segmentation/

## Techniques Used
- Exploratory Data Analysis (EDA)
- K-Means Clustering
- Hierarchical Clustering
- Elbow Method
- Silhouette Analysis
- Data Visualization

## Tools & Technologies
- R
- RStudio
- Quarto
- ggplot2
- factoextra
- dplyr
